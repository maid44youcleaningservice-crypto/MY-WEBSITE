MAID4YOU — PACOTE FINAL PARA GITHUB + VERCEL

Este pacote contém todos os arquivos curados para subir no GitHub e deixar a Vercel fazer o deploy automático.

Subir no GitHub:
1. Abra o repositório do site.
2. Apague/substitua os arquivos antigos.
3. Faça upload de TODOS estes itens na raiz do repositório:
   - index.html
   - quote.html
   - thankyou.html
   - privacy.html
   - cookies.html
   - pasta assets inteira

Muito importante:
- Não suba a pasta 'maid4you_github_vercel_final' como uma pasta dentro do repositório.
- Entre dentro dela e suba o conteúdo.
- A pasta assets precisa ir junto. Sem ela, logo, favicon e imagens podem quebrar.
- Depois do commit, a Vercel faz deploy automaticamente.
- O link da Vercel será o preview para enviar ao cliente.

Eventos/DataLayer:
- click_to_estimate: dispara ao abrir quote.html.
- click_to_text: dispara ao clicar em SMS/Text.
- form_submit: dispara ao abrir thankyou.html.
